Propriedade CSS: position

A propriedade position no CSS define como um elemento é posicionado na página. Ela permite controlar o fluxo e a sobreposição dos elementos no layout.

Os valores mais comuns para position são:

  • static: Padrão. O elemento segue o fluxo normal do documento.
  • relative: O elemento é posicionado em relação à sua posição original.
  • absolute: O elemento é posicionado em relação ao ancestral mais próximo que não seja static.
  • fixed: O elemento é posicionado em relação à janela de visualização (viewport), permanecendo fixo durante a rolagem.
  • sticky: O elemento alterna entre relative e fixed dependendo da rolagem.

Além disso, propriedades como top, right, bottom e left podem ser usadas junto com position para definir a posição exata do elemento.

Exemplo:

.exemplo {
    position: absolute;
    top: 20px;
    left: 50px;
}

Utilize a propriedade position para criar layouts mais flexíveis e controlar o empilhamento de elementos na sua página.
